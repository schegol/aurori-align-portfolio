Aurori Align: адаптивная верстка веб‑сайтов

Портфолио

Верстка всех макетов (кроме «Café Glitch») полностью совместима с Internet Explorer 11 и другими популярными браузерами. Адаптивные макеты могут комфортно использоваться на устройствах с любым размером экрана. Соблюдены принципы A11Y (доступность для людей с ограниченными возможностями, проработано управление с клавиатуры, прописаны нужные атрибуты для скринридеров) и прогрессивного улучшения (критический функционал, где возможно, доступен с выключенным JS). Верстка валидна, семантична.

Сайт приложения «Пинк»

Сайт приложения «Пинк»

Учебный проект HTML Academy, «Профессиональный HTML/CSS, уровень 2». Сверстаны все страницы и попапы для конкурсной формы. Раскладка блоков выполнена с помощью flexbox, адаптивная верстка под три разрешения в соответствии с концепцией «mobile-first». Именование классов по методологии БЭМ. Проведена ретинизация изображений, подключены их WEBP-версии с фоллбеком для браузеров без поддержки и векторный спрайт. Сборка проекта автоматизирована с помощью Gulp. В процессе работы использовался препроцессор Sass (синтаксис SCSS). Попапы и слайдер с отзывами на главной работают на чистом («ванильном») Javascript.

Сайт сметчика «Гарант Смета»

Одностраничный сайт с модальной формой. Раскладка блоков выполнена с помощью flexbox, адаптивная верстка (Bootstrap 4). Форма «Оставить заявку» и прочие интерактивные элементы работают на jQuery, анимированы с помощью jQuery и CSS. Функционирующие кастомные контактные формы (валидация Google reCaptcha и сбор данных с использованием технологии AJAX, отправка оповещения с собранными данными владельцу сайта с помощью PHP (PHP-сценарий отсутствует в репозитории Github из соображений безопасности)).

Интернет-магазин «Техномарт»

Учебный проект HTML Academy, «Профессиональный HTML/CSS, уровень 1». Сверстаны главная страница, страница каталога и интерактивные формы. Раскладка блоков выполнена с помощью flexbox, фиксированная верстка. Формы («Заблудились?», «Товар добавлен в корзину» и карта) работают на чистом JS, анимированы с помощью CSS. Слайдеры работают на чистом CSS.
